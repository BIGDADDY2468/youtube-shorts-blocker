# :bulb: YouTube Shorts Blocker

This is a Chrome extension that automatically redirects any YouTube Shorts page to the YouTube homepage.

## :question: Reason why this extension is used

The Chrome extension is used to ensure that the user is not able to access YouTube Shorts and therefore is not distracted while using YouTube. There are many people who waste a lot of time watching YouTube Shorts, and this is a distraction that ends up wasting a lot of time. This Chrome extension is able to detect the YouTube Shorts homepage and redirect the user to the YouTube homepage.

Through this, the user is able to stay focused on their work or studies and is able to avoid wasting time on YouTube Shorts. It is able to work in the background without the user's intervention. Overall, the purpose of this extension is to help the user stay productive and avoid wasting time on YouTube.

## 🚀 Features
- Detects YouTube Shorts URLs
- Automatically redirects to YouTube home
- Fast
- No user data collected

## 🛠️ Technologies Used
- JavaScript
- Chrome Extension Manifest V3

## 📂 Project Structure
yt-shorts-blocker : folder name

manifest.json : blueprint of the extension.

background.js : Background script.

## ⚙️ Installation
1. Download or clone this repository
2. Open Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click Load unpacked
6. Select the project folder

## 🎯 How It Works
This Chrome extension automatically detects any YouTube Shorts URL, i.e., youtube.com/shorts/, and redirects it to the homepage of YouTube.
